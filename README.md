Smart Meeting Scheduler

Description:
This project is an AI-based Smart Meeting Scheduler developed using FastAPI.
It helps users find suitable meeting times based on participant availability.

Features:
- User login simulation
- Participant availability input
- Automatic meeting time suggestion
- Web interface using HTML and CSS

Technologies Used:
- Python
- FastAPI
- HTML
- CSS

How to Run:
1. Install requirements:
   python -m pip install -r requirements.txt

2. Run server:
   python -m uvicorn main:app --reload

3. Open browser:
   http://127.0.0.1:8000
